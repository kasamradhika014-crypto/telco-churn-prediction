# Telecom Customer Churn Prediction

## Project Overview
This project predicts whether a telecom customer will churn using machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Building

## Outcome
The model helps identify high-risk customers and improve retention strategies.
